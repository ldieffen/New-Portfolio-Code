For a class we were all told to go buy a random object and then design a way to package it for retail. I went to a local thrift/junk shop and found this medal that caught my eye. I decided to package it as a fashion item for a company I named "Vintage Couture". Mocking this project up was a challenge, but I was very pleased of how it turned out in the end!
